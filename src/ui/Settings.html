📄 Sobre

HE - Online é um sistema completo e moderno para gerenciamento de históricos escolares, desenvolvido com tecnologias web modernas e hospedado em Google Apps Script.

Funcionalidades
  • ✅ Cadastro e gerenciamento de alunos
  • ✅ Controle de históricos escolares
  • ✅ Lançamento de notas e disciplinas
  • ✅ Emissão de certificados
  • ✅ Exportação de dados (CSV, PDF)
  • ✅ Dashboard com estatísticas
  • ✅ Interface responsiva e moderna
Tecnologias
  • Google Apps Script
  • Google Sheets
  • HTML5 / CSS3
  • Bootstrap 5
  • JavaScript Vanilla
Desenvolvedor

Desenvolvido como solução de gerenciamento educacional com padrão SIMADE/MG
